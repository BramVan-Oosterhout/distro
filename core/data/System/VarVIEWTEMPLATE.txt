#VarVIEWTEMPLATE
---+++ VIEW_TEMPLATE -- Specify VIEW [[%SYSTEMWEB%.SkinTemplates][SkinTemplate]] topic preference
   * This preference is used to specify a non-standard [[%SYSTEMWEB%.SkinTemplates][SkinTemplate]] with which to view a topic.
   * Example:
<verbatim>
      * Set VIEW_TEMPLATE = Web/SubWeb.SpecialViewTemplate
</verbatim>
