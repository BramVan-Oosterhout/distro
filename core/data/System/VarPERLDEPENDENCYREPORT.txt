#VarPERLDEPENDENCYREPORT
---+++ PERLDEPENDENCYREPORT -- report perl module dependencies
Intended for use by installers / administrators, reports on the status of
Perl dependencies in the Foswiki core and extensions.
   * Syntax: =%<nop>PERLDEPENDENCYREPORT%=
   * Supported parameters:
     | *Parameter:* | *Description:* | *Default:* |
     | ="extensions"= | requests dependencies of installed extensions. | (not set) |
   * See also: %SYSTEMWEB%.PerlDependencyReport
