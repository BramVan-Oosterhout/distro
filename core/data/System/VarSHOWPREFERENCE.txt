---+ SHOWPREFERENCE -- show where preferences are defined.
Preference values are shown in a bulleted list.
---++ Parameters
     %TABLE{sort="off"}%
     | *Parameter* | *Description* | *Default:* |
     | ="name"= | Name of preference to show | |
---++ Examples
   * =%<nop>SHOWPREFERENCE%= will show all preferences
   * =%<nop>SHOWPREFERENCE{"PREFERENCENAME"}%= will show a single preference
<verbatim class="tml">%SHOWPREFERENCE{"ATTACHFILESIZELIMIT"}%</verbatim>
      * Expands as: <pre class="tml">%SHOWPREFERENCE{"ATTACHFILESIZELIMIT"}%</pre>
   * =%<nop>SHOWPREFERENCE{"PREFERENCENAME,PREFERENCENAME,..."}%=
      * Show all of the preferences in a comma-separated list of preference names. This is particularly useful for reviewing the access controls that apply to a topic.
      * <verbatim class="tml">%SHOWPREFERENCE{"DENYWEBCHANGE,ALLOWWEBCHANGE"}%</verbatim> expands as: <pre class="tml">%SHOWPREFERENCE{"DENYWEBCHANGE,ALLOWWEBCHANGE"}%</pre>
